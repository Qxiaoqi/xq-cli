## xq-cli

简单的cli工具，后续会逐步扩展功能

## 使用

### 全局安装

```bash
npm i -g xq-cli
```

### 创建项目

#### webpack多页应用模板
[相关文档](https://github.com/Qxiaoqi/webpack-template)

```bash
xq init webpack <project-name>
```

#### 超简单的模板（可无视）
[相关文档](https://github.com/Qxiaoqi/simple-template)

```bash
xq init simple <project-name>
```

### 查看可用模板

```bash
xq list
```

### 查看帮助

```bash
xq --help
```
